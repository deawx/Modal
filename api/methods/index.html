frame ( options ) ( static )

frame() 方法是 Modal 对象的一个静态方法。它会创建一个包含 iframe 框架的弹出窗口。

Parameters

options：
Object - 配置信息
options.url：
String - iframe 要加载的页面地址。
options.scrolling：
[String] - iframe 是否出现滚动条（默认值：yes）。
options.title：
[String] - 提示框的标题文本 （默认值：窗口）
options.delay：
[Number] - 提示显示的时间 （默认值：-1），默认显示3秒后自动关闭。当值 options.delay <= 0，即表示必须调用 close() 方法手动关闭窗口。
options.buttons：
[Array] - 窗口的操作按钮配置信息。
options.width：
[Number] - 窗口的宽度（默认值：600）。
options.height：
[Number] - 窗口的高度（默认值：360）。
options.hasClose：
[Boolean] - 是否显示关闭按钮（默认值：true）。
options.hasHeader：
[Boolean] - 是否显示标题栏（默认值：true）。
options.afterClose：
[Function] - 窗口关闭后的回调函数。

Returns

loading() 方法是 Modal 对象的一个静态方法。它会创建一个操作处理中提示弹出窗口。

Parameters

String - 提示框的正文
options.hasHeader：
[Boolean] - 是否显示标题栏（默认值：false）。
options.title：
[String] - 提示框的标题文本 （默认值：加载中...）
options.delay：
[Number] - 提示显示的时间 （默认值：3000），默认显示3秒后自动关闭。当值 options.delay <= 0，即表示必须调用 close() 方法手动关闭窗口。
options.afterClose：
[Function] - 窗口关闭后的回调函数。

Returns

tip ( options ) ( static )

tip() 方法是 Modal 对象的一个静态方法。它会创建一个比 Modal.dialog 更加小巧的模拟窗口，标题栏和关闭按钮都更加小，而且是窗口显示几秒后是会自动关闭的。

Parameters

options：
Object - 配置信息
options.title：
String - 窗口标题文本
options.content：
String - 正文内容
options.hasHeader：
[Boolean] - 是否显示标题栏（默认值：true）。
options.title：
[String] - 提示框的标题文本 （默认值：窗口）
options.hasClose：
[Boolean] - 是否显示关闭按钮 （默认值：true）
options.hasOverlay：
[Boolean] - 是否有遮罩层 （默认值：true）
options.delay：
[Number] - 提示显示的时间 （默认值：3000）
options.width：
[Number] - 窗口宽度 （默认值：3000）
options.height：
[Number] - 窗口高度 （默认值：3000）
options.buttons：
[Array] - 窗口的操作按钮，如果没有设置，则不会显示按钮栏。
options.afterClose：
[Function] - 窗口关闭后的回调函数。

Returns

new Modal()：
Object - Modal 对象实例

[Number] - 提示显示的时间 （默认值：3000）
