Alert

 

Modal.alert()

效果图

Normal

Modal.alert('请填写完整的产品信息！','警告');

// 或者这么写，可以省略窗口标题
Modal.alert('请填写完整的产品信息！');

Close with callback

Modal.alert('请填写完整的产品信息！','警告', function(options,modal){
    alert(modal.attributes.title);
});

// 或者这么写
Modal.warning('请填写完整的产品信息！',function(options,modal){
    alert(modal.attributes.title);
});

API Details

Method: Modal.alert( tip, title, callback )

Modal.tipWarning()

Modal.tipWarning() 相对 Modal.alert() 方法，会创建一个更加纯粹的提示窗口。不会有遮罩层，可以没有标题栏，而切会自动关闭。

Normal without close button

效果图 - 没有关闭按钮

Modal.tipWarning({
    // 提示信息 - 必选
    tip: '请填写完整的产品信息！',
    // 宽度 - 可选（默认值：280）
    width: 300,
    // 高度 - 可选（默认值：140）
    height:80,
    // 显示时间 - 可选（默认值：3000 毫秒）
    delay: 3500
});

// 或者这么写：
Modal.tipWarning({
    // 必选
    tip: '请填写完整的产品信息！'
});

Normal without close button

效果图 - 带有关闭按钮

Modal.tipWarning({
    // 提示信息 - 必选
    tip: '请填写完整的产品信息！',
    // 宽度 - 可选（默认值：280）
    width: 300,
    // 高度 - 可选（默认值：140）
    height:80,
    // 显示时间 - 可选（默认值：3000 毫秒）
    delay: 3500,
    hasClose: true
});

// 或者这么写：
Modal.tipWarning({
    // 必选
    tip: '请填写完整的产品信息！'
});
