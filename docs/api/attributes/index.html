Attributes

parent

parent 选项用来设置 Modal 弹出窗口要显示的父容器的 DOM 节点。

Type：
String | HTMLElement
Default Value：
document.body

title

title 选项用来设置 Modal 弹出窗口显示的标题栏文本。

Type：
String
Default Value：
窗口

content

content 选项用来设置 Modal 弹出窗口显示的主题内容。

Type：
String
Default Value：
empty string

hasClose

hasClose 选项用来设置 Modal 弹出窗口是否显示关闭按钮。

Type：
Boolean
Default Value：
true

hasOverlay

hasOverlay 选项用来设置 Modal 弹出窗口是否显示遮罩层。

Type：
Boolean
Default Value：
true

autoDisplay

autoDisplay 选项用来设置 Modal 弹出窗口是否一开始就显示出来。

Type：
Boolean
Default Value：
true

beforeBuild

beforeBuild 选项用来设置 Modal 弹出窗口在创建 DOM 前的回调功能函数。

Type：
Function
Default Value：
null

afterBuild

afterBuild 选项用来设置 Modal 弹出窗口在创建 DOM 后的回调功能函数。

Type：
Function
Default Value：
null

beforeOpen

beforeOpen 选项用来设置 Modal 弹出窗口在显示前的回调功能函数。

Type：
Function
Default Value：
null

afterOpen

afterOpen 选项用来设置 Modal 弹出窗口在显示后的回调功能函数。

Type：
Function
Default Value：
null

beforeResize

beforeResize 选项用来设置 Modal 弹出窗口在调整尺寸前的回调功能函数。

Type：
Function
Default Value：
null

afterResize

afterResize 选项用来设置 Modal 弹出窗口在调整尺寸后的回调功能函数。

Type：
Function
Default Value：
null

beforeUpdatePosition

beforeUpdatePosition 选项用来设置 Modal 弹出窗口在重新定位前的回调功能函数。

Type：
Function
Default Value：
null

afterUpdatePosition

afterUpdatePosition 选项用来设置 Modal 弹出窗口在重新定位后的回调功能函数。

Type：
Function
Default Value：
null

beforeClose

beforeClose 选项用来设置 Modal 弹出窗口在关闭前的回调功能函数。

Type：
Function
Default Value：
null

afterClose

afterClose 选项用来设置 Modal 弹出窗口在关闭后的回调功能函数。

Type：
Function
Default Value：
null

buttons

buttons 选项用来设置 Modal 弹出窗口要显示的按钮（文字、按钮动作类别、按钮样式和按钮点击的回调函数）。如果没有设置，则 Modal 窗口不显示按钮栏。

Type：
Array
Default Value：
[]

TMPL_WRAP

TMPL_WRAP 选项用来设置 Modal 弹出窗口根节点的 HTML 模板代码片段。

Type：
String
Default Value：
<div class="modal-wrap ' + CLS_HIDE + '" id="modal-{id}"></div>

TMPL_HEADER

TMPL_HEADER 选项用来设置 Modal 弹出窗口按钮栏的 HTML 模板代码片段。

Type：
String
Default Value：
<div class="modal-header"></div>

TMPL_TITLE

TMPL_TITLE 选项用来设置 Modal 弹出窗口标题文本容器的 HTML 模板代码片段。

Type：
String
Default Value：
<h2 class="modal-title">{title}</h2>

TMPL_CLOSE

TMPL_CLOSE 选项用来设置 Modal 弹出窗口关闭按钮的 HTML 模板代码片段。

Type：
String
Default Value：
<div class="modal-close"><i class="icon-cross" title="'+ zh_CN.CLOSE +'"></i></div>

TMPL_BODY

TMPL_BODY 选项用来设置 Modal 弹出窗口内容容器的 HTML 模板代码片段。

Type：
String
Default Value：
<div class="modal-body"></div<

TMPL_CONTENT

TMPL_CONTENT 选项用来设置 Modal 弹出窗口内容容器的 HTML 模板代码片段。

Type：
String
Default Value：
<div class="modal-content"></div>

TMPL_INFOMATION

TMPL_INFOMATION 选项用来设置 Modal 弹出窗口内容文本容器的 HTML 模板代码片段。

Type：
String
Default Value：
<div class="modal-information"></div>

TMPL_FOOTER

TMPL_FOOTER 选项用来设置 Modal 弹出窗口按钮栏的 HTML 模板代码片段。

Type：
String
Default Value：
<div class="modal-footer"></div>

TMPL_BUTTON

TMPL_BUTTON 选项用来设置 Modal 弹出窗口操作按钮的 HTML 模板代码片段。

Type：
String
Default Value：
<button type="button" data-action="{action}" class="modal-button">{text}</button>

TMPL_OVERLAY

TMPL_OVERLAY 选项用来设置 Modal 弹出窗口遮罩层的 HTML 模板代码片段。

Type：
String
Default Value：
<div class="modal-overlay"></div>

delay

delay 选项用来设置 Modal 弹出窗口展示多少秒后自动关闭（单位：毫秒），默认是不会自动关闭的。

Type：
Number
Default Value：
0

width

width 选项用来设置 Modal 弹出窗口的宽度。

Type：
Number
Default Value：
600

height

height 选项用来设置 Modal 弹出窗口的高度。

Type：
Number
Default Value：
360
