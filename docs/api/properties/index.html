Properties

version

version 属性是 Modal 对象的当前版本。

attributes

attributes 属性用来存储 Modal 对象初始化时设置的配置项信息，详情请查看 Attributes 页面。

Type：
Object
Default Value：
Modal.defaults

CUSTOMEVENTS( static )

CUSTOMEVENTS 属性是 Modal 对象的静态属性，存储着 Modal 对象注册的所有合法的自定义事件名称。

Type：
Array

defaults( static )

defaults 属性是 Modal 对象的静态属性，存储着 Modal 对象初始化时需要的默认配置项信息，详情请查看 Attributes 页面。

Type：
Object

parent

parent 属性是 Modal 弹出窗口要插入的父节点的 DOM 元素。

Type：
HTMLElement
Default Value：
document.body

wrap

wrap 属性是 Modal 弹出窗口跟节点的 DOM 元素。

Type：
HTMLElement

header

header 属性是 Modal 弹出窗口标题栏的 DOM 元素。

Type：
Boolean

title

title 属性是 Modal 弹出窗口标题的 DOM 元素。

Type：
HTMLElement

closebar

closebar 属性是 Modal 弹出窗口关闭按钮的 DOM 元素。

Type：
HTMLElement

body

body 属性是 Modal 弹出窗口内容框的 DOM 元素。

Type：
HTMLElement

content

content 属性是 Modal 弹出窗口正文内容的 DOM 元素。

Type：
HTMLElement

information

information 属性是 Modal 弹出窗口文本内容的 DOM 元素。

Type：
HTMLElement

footer

footer 属性是 Modal 弹出窗口按钮栏的 DOM 元素。

Type：
HTMLElement

buttons

buttons 属性是 Modal 弹出窗口所有按钮 DOM 元素的数组。

Type：
Array

overlay

overlay 属性是 Modal 弹出窗口遮罩层的 DOM 元素。

Type：
HTMLElement

width

width 属性用来设置 Modal 弹出窗口宽度。

Type：
Number
Default Value：
600

height

height 属性用来设置 Modal 弹出窗口高度。

Type：
Number
Default Value：
360
